Import Process Overview

  1. Scan - Find files on disk not in the database
  2. Select - Choose which files to import
  3. Preview - Review how files will be parsed and check for duplicates
  4. Execute - Import the files with your chosen conflict resolutions

What gets imported:

  • Artist & Title - From ID3 tags (preferred) or filename parsing
  • Genre - Matched to existing categories, or "za obradu" if unknown
  • Decade - Calculated from year (e.g., 1985 -> 1980's)
  • Duration, Album, Year - From ID3 tags
  • Default settings - Enabled, Auto-DJ, Priority=5, etc.
